# Sample Videos

Place your highway video files here before running the detector.

## Expected filename for the default CLI example

```
samples/highway.mp4
```

## Acquiring sample footage

- [Pexels – free stock highway videos](https://www.pexels.com/search/videos/highway/)
- [Pixabay – traffic videos](https://pixabay.com/videos/search/traffic/)
- Any dashcam or traffic-camera MP4 / AVI file will work.

## Supported formats

`mp4`, `avi`, `mov`, `mkv`

> **Note:** Video files are excluded from version control via `.gitignore`.
> Only this README is tracked.
